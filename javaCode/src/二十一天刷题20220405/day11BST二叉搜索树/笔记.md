# 笔记
二叉搜索树BST（binary search tree）的特性很明显，左子树中的所有节点的值都小于根节点的值，右子树中的所有节点的值都大于根节点的值。

BST的递归算法模版：

```java
void BST(TreeNode root,int target){
    if(root.val==target){
        //做点什么    
    } 
    if(target<root.val){
        BST(root.left,target);    
    }
    if(target>root){
        BST(root.right,target);    
    }
}
```
